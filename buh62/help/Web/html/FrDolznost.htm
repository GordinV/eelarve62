Форма должность

Эта фориа преднозначена для определения должности .

Для того что бы определить должность надо заполнить поля подразделение - выбрать из списка подразделение к которому будет относиться должность, указать код должности, наименование и указать количество свободных мест. Остольные поля заполняются при необходимости.
