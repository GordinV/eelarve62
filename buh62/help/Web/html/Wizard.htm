Мастер

Мастер преднозначен для пересчета данных

Окно мастера состоит из двух частей. В первой части отображается информация по всем записям имеющимся в данном построителе, во втором выбранные записи. Для того что бы выбрать запись существуют специальные кнопки

Кнопка с изображением треугольника предназначена для отправки одной записи из списка в выбранные. Что бы проделать эту операцию необходимо предварительно выделить запись из списка и нажать на кнопку с изображением треугольника. Если всё было проделано правильно то запись окажется в списке выбранных

Кнопка с таким изображением

предназначена для отправления всех данных в список выбранных

Существуют так же две аналогичные кнопки только треугольники у них направлены в другую сторону. Эти кнопки выполняю аналогичные действия только в другую сторону

Для перехода на следующий шаг используются кнопки вперёд и назад расположенные в нижней части формы.

Что бы покинуть мастера нажмите на кнопку выхода
