Справочник остатки запасов на складе

Справочник остатки запасов на складе отображает информацию о имеющемся количестве таваров

Справочник состоит из пяти столбцов:

  • Код - определяет товар
  • Наименование - название товара
  • Группа - отображает в какую группу входит данный товар
  • Цена - стоимость за единицу
  • Остаток - количество товара имеющегося в распорижении

Контекстное меню содержит коменду перерасчет для того что бы вы могли пересчитать остаток если были внесены изменения

Работу с записями, а так же многое другое смотри в функциональных возможностях справочников
