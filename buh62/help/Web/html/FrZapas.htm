Форма запас

Эта форма преднозначена для формирования запаса.

Для формирования запаса необходимо заполнить следующие поля:

  • Код - код запаса
  • Наименоание - название запаса
  • Группа запасов - определяется путем выбора из разворачивающегося списка группы в которую будет входить данный запас
  • Ед. измерения - устанавливается еденица измерения для данного запаса
  • Цена - цена за еденицу
  • Минимальное количество - при помощи этого поля производится контроль за минимальным количеством запаса на складе. Когда установленная цифра превышена автоматически формируется заявка.
  • Наценка - в это поле устанавливается наценка на запас в процентах
  • Из трех разворачивающихся списков выбираются классфикаторы для бюджетных организаций
  • Примечание - коментарии
