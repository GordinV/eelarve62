Печать

Для того что бы отправить на печать документ необходимо в панели инструментов нажать на кнопку с изображением принтера, так же это можно сделать уже в открытом документе нажав на аналогичную кнопку

При этом появится окно печати

\

В раскрывающемся списке устанавливается объект печати.

В поле со счётчиком указывается количество экземпляров.
