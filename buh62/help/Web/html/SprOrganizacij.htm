Справочник организаций

Справочник организаций отображает информацию о имеющихся организациях

Справочник состоит из трех столбцов:

  • Регистрационный код - код организации
  • Наименование - название организации
  • Форма собственности

Контекстное меню содержит две характерные для этого справочника команды: сальдо и договора

При помощи команды сальдо вы можете открыть справочник сальдо для конкретной организации

Команда договора открывает справочник договоров для данной организации

Работу с записями, а так же многое другое смотри в функциональных возможностях справочников
