Форма карточка на работника

Эта форма преднозначена для для изменения личных нанных работника, трудовых договоров, а так же в начислений и удержаний

Форма состоит из трех вкладок: личные данные, трудовые договора, начисления и удержания.

Личные данные
На вкладке личные данные находится информация о работнике

Трудовые договора
Здесь отображается список трудовых договоров для конкретного работника

Начисления и удержания
На этой вкладке содержится список начислений и удерданий для данного работника

Списки с записями расположенные на вкладках трудовые договора и начисления и удержания имеют стандартные функции редактирования идентичные справочникам.

В правой части формы находится кнопка табель

которая открывает справочник табель рабочего времени для данного работника
