Форма ведомость

Эта форма преднозначина для создания начислений или удержаний для конкретного работника.

Для создания нового начисления необходимо выбрать из справочника ведомость на зар.плату работника для которого будет создано начисление, затем при помощи имебщихся возможностей (контекстное, дополнительное меню или панель инструментов) создать новую запись. В открывщейся форме надо заполнить поле сумма - указать сумму начисления. Остольные поля заполняются автоматически, но при желании можно изменить их значение. На форме расположены четыре обязательных поля: должность - выбирается должность работника, код - код начисления, дата и сумма. Поле примечание используется для коментариев.
