Главное меню

Главное меню это средство управления программой которое находится сразу под заголовком окна

Главное меню состоит из следующих пунктов

  1. Ввод данных
  2. Редактирование
  3. Отчёты
  4. Справочники
  5. Настройка 
  6. Операции

Иногда появляется дополнительный пункт главного меню в зависимости от того какое окно открыто


В данном примере это окно журнал операций
