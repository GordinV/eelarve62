Форма подготовка отчетов

Эта форма преднозначена для формирования отчета с последующем выходом на печать

В верху формы находится разворачивающийся список из которого выбирается нужный отчет, затем устанавливается срок по которому будет сформирован отчет. Все поля и списки находящиеся ниже относятся к конкретному отчету и становятся октивными только если ввод информации в эти поля предусмотрен для данного отчета.

В правой части формы находятся три функциональные кнопки

Верхняя кнопка с изображением синий галочки преднозначена для формирование отчета и отправки его на печать. Кнопка с изображением красного крестика закрывает форму. Кнопка настройка открывает форму с настройками прграммы.
