Журнал хозяйственных операций

Журнал хозяйственных операций это справочник в котором отображаются существующие операции

Журнал имеет структуру таблицы и состоит из восьми столбцов:

  • Дата - показывает дату для каждой операции
  • Дебит - отображает код по дебету
  • Кредит - показывает код по кредиту
  • Сумма - сумма каждой операции
  • Пояснение - описание операции
  • Документ
  • Партнер - наименование партнера
  • Проводка - отображает код проводки

Журнал хозяйственных операций имеет функцию добавленную в контекстное и в дополнительное менбю позволяющую находить запись по номеру проводки

Для того что бы найти запись по номеру проводки необходимо выбрать из контекстного или дополнительного меню соответствующую команду. Перед вами появится окно позволяющее ввести в текстовое поле номер проводки.

Когда номер введен нажмите на кнопку ввода и если заданный вами номер существует то тогда откроется форма операция в режиме редактирования

Редактирование и удаление записей, а так же многое другое смотри в функциональных возможностях справачников
