Форма закрытие периода

Эта форма преднозначена для закрытия периода на определенную дату

Эта форма состоит из текстового поля дата: в котором указывается за какую дату будет закрыт период, проводка по доходам: это разворачивающийся список из которого нужно выбрать тип проводок по доходам, проводки порасхода: указывается тип проводок по расходам.

Если разворачивающиеся списки пусты то тогда надо добавить расходные и доходные проводки в журнал стандартных операций. При этом тип документа должен быть обозначен как Period.
