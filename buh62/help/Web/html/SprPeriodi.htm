Справочник периоды

Справочник периоды преднозначен для закрытия периода

Справочник состоит из четырех столбцов:

  • Год - отображает год
  • Месяц - показывает месяц
  • Закрытие - закрыт период или нет
  • Текущий - является ли период текущим

На форме расположены три функциональные кнопки. Кнопка с таким изображением

преднозначена для закрытия периода

Эта кнопка формирует проводки для данного периода. При нажатии на эту кнопку открывается окно формы формирования проводок

Контекстное меню состоит из трех команд: изменить статус, печать и сформировать проводки
