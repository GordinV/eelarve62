Форма праздник

Эта форма преднозначена для назначения праздничных дней

Она состоит из четырех полей

  • День - вводится день праздника цифрами
  • Месяц - указывается месяц
  • Наименование - название праздника
  • Примечание - преднозначено для коментариев

Создание праздников необходимо при подсчете рабочего времени работника.
