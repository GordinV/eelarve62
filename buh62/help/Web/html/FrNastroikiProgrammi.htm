Форма настройка программы

Эта форма состоит из восьми вкладок. Изменять данные в этой форме рекомендуется только опытным пользоватедям или админестраторам.

Основные данные

Здесь можно изменить общие настройки такие как язык, установить картинку на рабочее поле: для этого надо нажать на кнопку с тремя точками затем из стандартного окна Windows выбрать нужную картинку, так же можно установить округление которое будет отображаться во всех счетах

Электронная почта

На вкладке электронная почта настраиваются данные касающиеся электронной почты

Следующие четыре вкладки настраивабтся при установки или администратором баз данных

Зарплата

На этой вкладке можно установить минимальную заработанную плату, сумму не облогаемаю налогом и степень округления.

Доходы/Расходы

Здесь формируется аналитика по доходам и расходам.
