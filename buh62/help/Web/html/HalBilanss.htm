Настройка баланса

Бухгалтерский баланс формируется на основе шаблона, содержащего строки баланса и формул - сальдо групп счетов

В колонке Rea nr указывается строка баланса. При расшифровки обязательно наличие основной строки. Напр. (1 Деньги. 1.1 Касса)

В колонку Nimetus указывается наименование строки

В колонке Formula указывается формула.

Поддерживаются следующие формулы: SD(ГРУППА СЧЕТОВ) сальдо по дебету, SK (ГРУППА СЧЕТОВ) сальдо по кредиту, KK(ГРУППА СЧЕТОВ) оборот по кредиту и DK (ГРУППА СЧЕТОВ) оборот по кредиту. Формулы можно комбинировать с основными математическими операциями +, -, *, /. Например SD(191)+SD(192) суммируются остатки групп счетов 191 и 192
