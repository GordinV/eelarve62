Журнал

Форма журнал предназначена для отображения хода выполнения сметы на определённый момент времени.

В поле дата вносится дата создания записи в журнале.

Записи в журнал можно добавить или удалить при помощи контекстного меню щелкнув правой кнопкой по записи и выбрав соответствующий пункт.
