Форма организация

Эта форма преднозначена для определения организации которая будет в дальнеешем использована при работе с базой данных. В этой форме вы можете установить свойства характерные для данной организации. Для этого необходимо заполнить поля код, форма собственности и наименование организации. Остольные поля являются необязательными и заполняются на усмотрение пользователя. Находящаяся в правой части формы кнопка сальдо

позволяет пользователю определить сальдо для данной организации

смотри так же редактирование формы
