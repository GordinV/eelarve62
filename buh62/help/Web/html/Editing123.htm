Редактирование записей

Редактирование при помощи мыши
- двойной щелчок:
 для того что бы отредактировать уже существующую запись при помощи мыши необходимо дважды щелкнуть на этой записи. Тогда эта запись откроется в режиме редактирования
- вызов контекстного меню:
для вызова контекстного меню надо нажать правой кнопкой мыши на записи
- вызов главного меню:
вызов главного меню редактирования производится при выборе правой кнопкой мыши меню редактирования
- вызов вспомогательного меню:
вспомогательное меню редактирования появляется при запуске конкретного справочника и присваивает имя открывшегося справочника

Редактирование при помощи клавиатуры
- клавиша Enter:
Если запись которую вы хотите отредактировать активна то тогда вы можете открыть ее нажав на кнопку Enter
- быстрые клавиши:
существует возможность редактирования записей используя клавиатурные комбинации

В пункте меню редактирование представлены стандартные команды

  1. Поиск - команда вызывается для того что бы найти какую то конкретную запись
  2. Копировать - команда копирует выделенный объект
  3. Вырезать - команда вырезает выделенный объект
  4. Вставить - команда вставляет скопированный или вырезанный объект в соответствии с расположением курсора
  5. Отмена след. операции
  6. Отмена посл. операции

В вспомогательном меню (а так же в дополнительной панели инструментов) представлены следующие команды

  1. Удалить - удаляет запись.
  2. Создать новую - создает новую запись
  3. Внести изменения - открывает запись в режиме редактирования
  4. Печать - Открывает окно печати
  5. Обновить - Обновляет активное окно
