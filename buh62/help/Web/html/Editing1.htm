Редактирование формы

Принцип редактирования является общим для всех форм базы данных

При открытии формы активное поле становится красным. Если поле активно то тогда вы можете вносить изменения в это поле. При редактировании формы действуют команды редактирования главного меню редактирования.

Для того что бы внести сделанные изменения необходимо нажать на кнопку с изображением зелёной галочки

тогда ваши изменения будут приняты.

Для того что бы отказаться от изменений необходимо нажать на кнопку с изображением красного крестика

Работа с текстовым полем:

При переходу к текстовому полю оно становится активным и выделяется красным цветом. Информация вводится в текстовое поле при помощи клавиатуры. Если информация введена то для перехода на другое поле можно использовать клавишу ENTER или TAB. При нажатии правой кнопкой мыши на текстовом поле открывается контекстное меню с указанными в нем пунктами редактирования

Работа с разворачивающимся списком:

Разворачивающийся список открывается при нажатии на стрелочку левой кнопкой мыши или при помощи кнопки F4. При двойном щелчке на разворачивающемся списке открывается форма на добавление элемента. Это можно проделать при помощи контекстного меню. Если вы вводите в поле со списком информацию которая не содержится в этом списке, то тогда после перехода на другое поле будет открыта форма на добавление элемента (некоторые списки не поддерживают эту функцию).

Работа с полем дата и кнопкой календаря:
Дату можно задать как при помощи клавиатуры, так и при помощи специальной кнопки расположенной рядом с соответствующим полем

При нажатии на эту кнопку открывается календарь в котором легко можно установить нужную дату.


Работа с закладками.
Для тогочто бы перейти на другую закладку необходимо левой кнопкой мыши щелкнуть на названии этой закладки.
