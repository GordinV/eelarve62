Форма счет на оплату

При помощи данной форы создается счет на оплату

При создании счета необходимо указать дату, выбрать из разворачивающегося списка платильщика и указать расчетный счет

Ниже находится таблица в которой можно определить компоненты счета

  • Код - из этого разворачивающегося списка выберается услуга которая войдет в счет
  • Цена - в это поле вводится цена за выбранную услугу
  • Количество
  • Без налога - в этом поле отображается стоимость без налога
  • Скидка - в поле скидка вводится стоимось скидки для данной услуги (если есть)
  • НСО - отображает налог с обарота с учетом стоимости услуги
  • Сумма - показывает общую сумму по данной услуге

В самом низу формы расположены поля: срок - устанавливается дата погашения счета, поля итого показывают сумму для расположенных над ними столбцов (являются результирующими для столбцов скидка, НСО и сумма)

В правой части формы расположены пять кнопок: договор, оплата, примечание на строку, примечание для документа и тип документа.

Кнопки примечаний служат для вызова блокнота в который можно записывать коментарии

Копка оплата дает доступ к форме оплата счета, тип документа позволяет выбрать тип документа

Редактирование записей, а так же работу с элементами формы смотри здесь
