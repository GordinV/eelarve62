Форма данные предприятия

Форма преднозначина для формирования предприятия и его данных.

Для создания нового предприятия вам необходимо заполнить поля регистрационный код предприятия, наименование, указать адрес и руководителя. Остольные поля являются необязательными

смотри так же редактирование формы
