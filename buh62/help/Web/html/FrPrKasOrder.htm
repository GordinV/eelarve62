Форма приходный кассовый ордер

Эта форма преднозначина для создания документа о получении денег.

Для того что бы создать новый приходный кассовый ордер необходимо указать номер приходного ордера, дату, выбрать из разворачивающегося списка платильщика. Остольные поля указываются при необходимости. Расположенная ниже таблица должна содержать компоненты приходного ордера: из разворачивающегося списка надо выбрать код и указать сумму. Под столбцом сумма расположено разультирующее поле в котором показывается общая сумма
