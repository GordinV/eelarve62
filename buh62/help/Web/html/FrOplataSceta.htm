Форма оплата счета

эта форма преднозначена для оплаты счета

Для того что бы счет был оплачен необходимо заполнить следующие поля

  • Счет номер: - разворачивающейся список из которого необходимо выбрать номер счета
  • Дата - указывается дата оплаты
  • Сумма - стоимость оплачиваемой услуги
  • Банк/касса - группа переключателей определяющая как будет производиться оплата
  • Операция - из этого списка выберается операция
  • Документ - вид документа
  • Примечание - коментарии

Справа на форму помещены две кнопки. Это ордер и тип документа. Кнопка ордер становится активной только когда поставлен переключатель касса. эта кнопка позволяет открыть форму операция. При помощи кнопки тип документа открывается одноименная форма.
