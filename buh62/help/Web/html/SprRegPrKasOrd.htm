Регистр приходных кассовых ордеров

Регистр приходных кассовых ордеров это справочник в котором отображаются приходные ордера

Регистр имеет структуру таблицы и состоит из пяти столбцов:

  • Номер - показывает номер кассового ордера
  • дата - показывает дату для каждого приходного кассового ордера
  • Платильщик - отображает платильщика
  • Сумма - показывает стоимость
  • Касса - наименование кассы

Редактирование и удаление записей, а так же многое другое смотри в функциональных возможностях справачников
