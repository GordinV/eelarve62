Начало работы

Программа "Смета" состоит из главного окна, главного меню и кнопок быстрого вызова. Главное окно располагается на самом верху экрана и содержит название программы, сразу под ним расположено главное меню доступ к которому осуществляется посредством нажатия на пункт меню кнопкой мыши или при нажатии на клавишу ALT пункт меню становится активным, для того что бы открыть это меню используется клавиша ENTER, для перемещения используйте клавиши управления курсором. В главном меню напротив пункта ниспадающего меню подписаны быстрые клавиши для вызова этой команды или элемента программы (список быстрых клавиш смотри ниже). Кнопки быстрого доступа обычно располагаются ниже главного меню и служат для вызова определенного справочника программы.
  • Для начала работы надо выбрать соответствующий пункт из главного меню

       

  • Можно использовать для этого так же и панель инструментов
