Форма операция

Форма операция служит для добавления новой или изменения старой операции

При создании новой операции в поле дата устанавливается по умолчанию текущая дата, их списка партнеров выбирается нужный партнер. В поле пояснение заносится текст который будет отображаться в журнале хозяйственных операций. В поле документ вводится наименование документа. Список признак служит для дополнительной группировки проводок по признаку. Dok.Liik определяет тип документа связанного с определенными проводками.

Ниже находится таблица в которой можно определить проводки для данной операции. Для этого надо выбрать из списка наименований проводок нужную, поля дебит и кредит заполняются автоматически, в поле сумма указывается сумма.

В правой части формы расположены три кнопки: Стандартные операции, в журнал стандартных операций и примечание.

При нажатии на кнопку стандартные операции открывается справочник стандартных операций из которого можно выбрать уже существующую операцию, если вы хотите добавить операцию то тогда необходимо установить курсор на нужную запись и закрыть окно. При этом появится диалоговое окно в котором вы должны подтвердить свой выбор.

Кнопка примечание служит для вызова блокнота в который можно записывать коментарии по поводу этой операции

Редактирование записей, а так же работу с элементами формы смотри в здесь
