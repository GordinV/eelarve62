Функциональные возможности справочников

Справочник представляет собой таблицу

Перемещение по справочнику производится при помощи клавиатуры и мыши. Если вы хотите перемещаться по записям при помощи клавиатуры то для этого вам необходимо использовать клавиши управления курсором. Для перемещения по записям при помощи мыши вам необходимо просто щелкнуть на нужной записи левой кнопкой мыши.

Редактирование записей:
1. При помощи мыши:
- двойной щелчок по записи
- вызов контекстного меню
- вызов главного меню редактирования
- вызов вспомогательного меню

2. При помощи клавиатуры:
- быстрые клавиши
- клавиша Enter
Подробнее о редактировании

Сортировка записей:
Для каждого столбца таблицы определен свой заголовок который находится выше всех имеющихся записей. Для того что бы отсортировать записи по определенному столбцу необходимо щелкнуть на нужном заголовке правой кнопкой мыши.

Фильтрация записей
Фильтры находятся в самом низу таблицы

Фильтр можно задать для каждого столбца таблицы при помощи ввода в ячейки цифрового или буквенного значения

В ячейки с кнопкой можно задавать интервал 

Для этого надо нажать на значок с кнопкой, при этом появится окно

В это окно вводится значение от и до

Переводчик
Функция позволяющая работать с несколькими языками
Подробнее о переводчике

Поиск
Встроенная функция позволяющая быстро найти нужную запись

Для того что бы воспользоваться этой функцией необходимо

  • выбрать в главном меню пункт редактирование, затем выбрать поиск

при этом появится окно поиска.

  • также эту команду можно вызвать при помощи кнопки F1
  • эта команда так же отображена в панели инструментов

Печать
При помощи функции печать можно распечатать весь список записей или конкретную запись
