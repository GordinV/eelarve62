Справочник основные средства

Справочник основные средства отображает информацию о основных средствах

Справочник состоит из восьми столбцов:

  • Инвентарный номер - отображает код
  • Наименование - название средства
  • Группа - отображает в какую группу входит данное средство
  • Счет - номер счета
  • Цена
  • Дата приобритения
  • Ответственное лицо - отображает лицо ответственное за данное средство
  • Списано - дата списания

Контекстное меню содержит команду расчет износа. При активизации этой команды вам будет предложено установить число и выбрать операцию. После чего откроется мастер расчета износа. При добавлении записи открывается форма в которой надо определить группу основных средств.

Работу с записями, а так же многое другое смотри в функциональных возможностях справочников
