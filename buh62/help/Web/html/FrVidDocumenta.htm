Форма вид документа

Форма вид документа преднозначена для создания или редактирования вида документа

Эта форма состоит из трех текстовых полей: код - указывается код, наименование - вводится наименование и примечание - коментарии относительно этого документа

Справа располагаются две кнопки: поводки и свойства

При выборе команды поводки открывается мастер при помощи каторого вы можете дабавить нужные проводки для данного документа

Команда свойства открывает форму документ где настраиваются свойства проводки

смотри так же редактирование формы
