Панель Инструментов

Панель инструментов состоит из семи основных кнопок

  1. Регистр смет
  2. Расценки
  3. Заказчики
  4. Справочник материалов
  5. Объекты
  6. Центры затрат
  7. Курс валют

При нажатии на одну из этих кнопок появляются дополнительные кнопки редактирования

  1. Добавить запись
  2. Редактировать
  3. Поиск
  4. Печать
  5. Удалить

Панель инструментов может быть перемещена в другое место. Для того что бы переместить панель инструментов необходимо зажать кнопку мыши на свободном от кнопок месте и не отпуская копки мыши переместить ее на удобное для вас место, затем отпустить кнопку мыши. Существует четыре стандартных расположения панели инструментов. По краям окна: сверху, снизу справа и слева. Когда вы перемещаете панель к одной из указанных позиций она автоматически становится на указанное место.
