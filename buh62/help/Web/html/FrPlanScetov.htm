Форма план счетов

Эта форма состоит из трех вкладок: счет, информация о счете и суб.счета.

Счет
На вкладке счет необходимо данные счета. Заполнить поля код, наименование счета и примечание (необязательно).

Информация о счете

На вкладке информация о счете необходимо указать тип счета и определить сальдо на начало года.

Типы счетов

При помощи переключателей определяется тип счета

СД - сальдо по дебету переносится на начало года

СК - сальдо по кредит переносится на начало года

Д - сальдо по дебету открывается и закрывается в течении периода

К - сальдо по кредиту открывается и закрывается в течении периода

ДК - сальдо по дебету и по кредету

В таблице расположенной ниже типов счетов указывается сальдо на начало года. Если сальдо кредитовое то его значение должно указываться со знаком минус.

Суб. счета

На вкладке суб. счета учитывается сальдо для организаций выступающих контр агентами. Учитывающиеся на данном счете организации попадают в этот список при наличии проводки. Сальдо можно отредактировать при условии открытого первого месяца.
