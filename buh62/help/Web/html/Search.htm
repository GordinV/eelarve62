Поиск

Встроенная функция позволяющая быстро найти нужную запись

Для того что бы воспользоваться этой функцией необходимо

  • выбрать в главном меню пункт редактирование, затем выбрать поиск

при этом появится окно поиска.

В ячейку выделенную красным цветом нужно ввести текст для поиска. Что бы активизировать команду следует нажать кнопку с изображением бинокля. Если запись найдена то она становится активной. Если вам нужно произвести поиск в обратном направлении, то для этого необходимо установить соответствующую галочку. когда запись не найдена появляется сообщение.

  • также эту команду можно вызвать при помощи кнопки F1
  • эта команда так же отображена в панели инструментов
