Форма табель рабочего времени

Эта форма преднозначена для формирования рабочего времени для конкретного работника.

Для того что бы определить рабочее врамя необходимо заполнить поля договор - выбрать из разворачивающегося списка код договора затем ввести месяц и год (эти поля устанавливаются автоматически) и указать количество часов в день. Осталные поля являются не обязательными и заполняются при необходимости.
